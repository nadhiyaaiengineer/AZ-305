Azure Virtual Machines (AZ-305)

Restaurant Analogy

CPU=Chef 👨‍🍳
RAM=Kitchen Counter
Disk=Store Room
GPU=Special Cake Decorating Chef 🎂

VM Families

  • B: Dev/Test
  • D: General Purpose
  • E: Memory Optimized
  • F: Compute Optimized
  • L: Storage Optimized
  • M: Large Memory
  • N: GPU

Scale Set

Like a restaurant franchise that automatically opens/closes branches based on customer demand.

Availability Set vs Zone

Availability Set: Different buildings in the same area (protects against rack/maintenance).

Availability Zone: Different parts of the city (different datacenters in same region).

Memory Trick

Chef=CPU, Counter=RAM, Store=Disk, Receptionist=Load Balancer.
