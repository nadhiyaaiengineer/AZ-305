Describe AI, Machine Learning and IoT/Edge Services in Azure

Overview

Azure offers intelligent services so you don't have to build everything from scratch.

Need pre-built AI?      → Azure AI Services
Need Generative AI?     → Azure OpenAI Service
Need custom AI models?  → Azure Machine Learning
Need connected devices? → Azure IoT Services

Azure AI Services

Ready-made AI capabilities exposed through APIs. No model training required.

  • Vision - Analyze images
  • Speech - Speech ↔ Text
  • Language - Translation, sentiment analysis
  • Document Intelligence - Extract information from forms and PDFs

Example

Passport Image
      ↓
Azure AI Document Intelligence
      ↓
Extract Name, DOB, Passport Number

Azure OpenAI Service

Provides Generative AI models for chat, content generation, summaries and code generation with Azure security and governance.

User Prompt
      ↓
Azure OpenAI
      ↓
Chat • Code • Summary • Email • Content

Agentic AI

Agentic AI is an application pattern, not a separate Azure service. It combines AI models, instructions, context and tools to accomplish multi-step goals.

User Request
      ↓
AI Understands Goal
      ↓
Search Database
      ↓
Call APIs
      ↓
Generate Response
      ↓
Complete Task

Example: "Book my flight" → Search flights → Compare prices → Book ticket → Email confirmation.

Azure Machine Learning

Used when you need to build, train, deploy and manage your own machine learning models.

Training Data
      ↓
Train Model
      ↓
Deploy Model
      ↓
Predictions

Example: Train a fraud detection model using your company's transaction history.

Azure IoT Services

IoT Hub

Secure bi-directional communication between devices and Azure.

IoT Central

A SaaS platform for quickly building IoT solutions with dashboards and device management.

IoT Edge

Runs workloads close to devices for faster local processing.

Sensors
   ↓
IoT Hub
   ↓
Cloud Analytics
   ↓
Model Updates
   ↓
IoT Edge
   ↓
Local Processing
